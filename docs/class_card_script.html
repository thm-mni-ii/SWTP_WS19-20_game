void CardScript.TimeUP ( )

This method destroys a "Card" tagged gameobject. If the game is neither in the votePhase nor is an answer given by the player, the SetCardFromPlayerScript method and the RegisterAnswer method are called. This method calls the CmdAnswerInc via PlayerScript ps, so that it will be executed on the Server.
